# tmux学习

```
执行tmux，进入tmux新环境
ctrl+b d，退出tmux环境，下次依旧可以进入到这些会话中
tmux a，进入tmux环境

t  窗口中央显示一个数字时钟
```

## 1 会话
```
ctrl+b s，列出所有会话
ctrl+b :new，新建一个会话
ctrl+b $，给会话重命名
```

## 2 窗口
```
ctrl+b w，显示当前会话中有几个窗口
ctrl+b c 新建一个窗口
ctrl+b , 重命名当前窗口
ctrl+b & 关闭当前窗口
```

## 3 窗格
```
ctrl+b % 垂直分割
ctrl+b " 水平分割
ctrl+b o 换窗格焦点
ctrl+b q 显示窗格数字
ctrl+b z 切换当前最大化/最小化
```
